# Gallery Images

This directory contains images for the gallery.

## Upload via Sveltia CMS

The easiest way to add images is through the CMS at `/admin`.

## Manual Upload

You can also manually add images here:

- `/public/images/sprchovekouty/`
- `/public/images/obklady/`
- `/public/images/dvere/`
- `/public/images/zabradli/`
- `/public/images/montaze/`
- `/public/images/opracovani/`

## Image Optimization

- Max width: 2000px
- Format: WebP or JPEG
- Quality: 85%
- File size: < 500KB per image
